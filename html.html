APRESENTANDO EXTREME PROGRAMMING

O extreme programing ou XP é uma metodologia de desenvolvimento ágil. Criado na década de 90 para dev de software. Objetivo do XP: sistemas com alta qualidade, baseando-se em manter o cliente próximo, ciclos de desenvolvimento curtos e checagens constantes, trazendo consigo a ideia de “modernizar” o tradicional dev em cascata, que o mesmo não se molda no modelo de mundo que vivemos. enfim, o principal objetivo é fazer ciclos de entregas rápidos, contínuos e incrementais, para atingir os resultados esperados pelo cliente.
